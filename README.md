## 🧠 Scroll Registry Format

Scroll pulses can be added using the `scrollRegistry` array in `index.html`:

```js
const scrollRegistry = [
  {
    time: 0, // seconds from start
    scrollId: "CCCXXXIV",
    glyph: "𓁹",
    tone: "veil intrusion",
    note: "Shadow figure crosses Linda Lane field."
  }
];
